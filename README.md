### ✨ Key Features

- 🎯 Destination-based seat allocation
- 📅 Date-wise booking management
- 🚫 Double-booking conflict detection
- 💾 File-based data persistence
- ✅ User input validation
- 🔄 Structured program flow
- 🛠️ Multi-feature console interface
